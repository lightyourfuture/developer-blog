## TypeScript 示例

TypeScript 提供了类型安全：

```typescript
interface User {
  name: string;
  age: number;
  email: string;
}

const user: User = {
  name: "张三",
  age: 25,
  email: "zhangsan@example.com"
};

console.log(`用户 ${user.name} 的年龄是 ${user.age}`);
```
